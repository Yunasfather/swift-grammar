# Day 02 - Numbers

## 학습 내용
- 정수 타입
  - `Int`, `UInt`, `Int8`, `Int16`, `Int32`, `Int64`
  - `.max`, `.min` → 각 타입의 범위
- 수 체계 & 진법
  - 2진수(`0b`), 8진수(`0o`), 16진수(`0x`)
  - `String(number, radix: N, uppercase:)` → 원하는 진법 문자열 변환
  - radix ≤ 10 → uppercase 옵션 영향 없음  
  - radix ≥ 11 → uppercase 옵션에 따라 알파벳 대/소문자 결정
- 바이트 단위
  - 1 byte = 8 bit
  - 1 KB = 1,024 byte
  - 1 MB = 1,024 KB
  - 1 GB = 1,024 MB
  - 1 TB = 1,024 GB
- 수 표현
  - 숫자 구분자: `1_000_000`
  - 나머지 연산: `%`
- 부동소수점
  - 2진수 기반이라 `0.7 + 0.2` 계산 시 오차 발생
  - `Decimal`을 사용하면 10진수 체계로 비교적 정확히 계산

## 연습 문제
1. `Int8`, `UInt8`, `Int16`, `Int32`, `Int64`의 최소값과 최대값을 출력하세요.  
2. `255`를 2진수, 8진수, 10진수, 16진수로 변환해 출력하세요.  
3. `123456`을 36진수로 변환해 lowercase/uppercase 버전을 출력하세요.  
4. 50을 2진수, 8진수, 16진수 리터럴로 선언해 출력하세요.  
5. `10 % 4`의 결과를 출력하세요.  
6. `0.7 + 0.2`를 Double과 Decimal로 각각 출력해 보세요.
