# Day 07 - Closures

## 학습 내용

### 1) 클로저 기본
- 함수에서 이름을 제거한 형태
- `{ (param) -> ReturnType in ... }`

### 2) 클로저 사용
- 변수/상수에 할당
- 함수의 인자로 전달

### 3) 후행 클로저
- 마지막 파라미터가 클로저일 경우, 함수 괄호 바깥에 작성 가능

### 4) 클로저 단축 문법
- 매개변수 생략: `$0`, `$1` …
- `return` 생략 가능 (단일 표현식인 경우)

### 5) 함수 + 클로저 조합
- 콜백(callback) 형태로 자주 활용
- 예: 네트워크 요청 완료 시 실행할 동작 전달

## 연습 문제
1. `(Int, Int) -> Int` 형태의 클로저를 선언해 두 수를 더한 결과 출력  
2. 문자열을 받아 `"Hello, ___!"`로 출력하는 클로저 작성  
3. `calculate(a:b:completion:)` 함수를 작성하고,  
   - 합계를 출력하는 클로저  
   - 합계가 10보다 큰지 출력하는 클로저  
   두 가지로 호출  
4. 후행 클로저 문법으로 배열 `[1,2,3,4,5]`의 짝수만 출력
