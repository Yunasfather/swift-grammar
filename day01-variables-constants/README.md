
# Day 02 - Strings & Booleans

## 학습 내용
- 문자열 프로퍼티 & 메서드
  - `count`, `isEmpty`
  - `+` (문자열 연결)
  - `uppercased()`, `lowercased()`, `capitalized`
  - `dropFirst()`, `dropLast()`
  - `hasPrefix()`, `hasSuffix()`, `contains()`
- Bool 연산
  - `!` (NOT)
  - `.toggle()`

## 연습 문제
1. 문자열 두 개를 연결해서 출력하세요. → `Hello, Swift!`  
2. 빈 문자열을 만들고 `isEmpty` 값을 출력하세요. → `false`  
3. 문자열 `"abc"`의 글자 수를 출력하세요. → `3`  
4. `"apple"`을 대문자로 출력하세요. → `APPLE`  
5. Bool 변수에 대해 `!`와 `toggle()`을 활용해 출력하세요. → `false`, `true`  
6. `"hello world"`를 변환해서 출력하세요.  
   - `Hello World`  
   - `hello world`  
   - `Hello world`  
7. `"Hello Swift"`에서 앞의 3글자를 제거하고 출력하세요. → `lo Swift`  
8. `"Swift Programming"`이 접두사/접미사/포함 여부를 확인하세요.  
   - 시작: `true`  
   - 끝: `false`  
   - 포함: `true`  
9. Bool 변수를 `toggle()` 두 번 적용해 출력하세요. → `true`, `false`
